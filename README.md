Another yet an awesome boilerplate to complement what already exists :-)


## Prerequisites

- [Docker](https://docs.docker.com/engine/installation/)

## Install

```sh
make install
```

## Dev

```sh
make run
```

### Create model
```sh
make generate-model MODEL=Foo
```

### Running migrations
```sh
make migrate
```

## Test

```sh
make test
```
